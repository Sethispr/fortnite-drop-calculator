# Fortnite Drop Calculator

Find the fastest way to land in Fortnite. This site calculates exactly when you should jump from the bus and when you should deploy your glider.

## How to use?

1.  **Set the bus path**: Drag the **S** (Start) and **E** (End) markers on the map to match the bus path.
2.  **Pick your drop spot**: Click the map or drag the **T** (Target) marker to where you want to land.
3.  **Follow the points**:
    *   **Jump (J)**: The best spot to leave the bus.
    *   **Deploy (D)**: The best spot to open your glider.

## Developing

If you want to run this project locally

- Install deps: `npm install`
- Start the app: `npm run dev`
- Build prod site: `npm run build`

You can also fork this repository and host + build the site with Github Pages.
